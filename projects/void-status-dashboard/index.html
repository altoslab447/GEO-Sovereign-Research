3. 台灣通訊對接 (LINE Guide)

LINE Messaging API 保姆級教程

1

Step 1: 建立管理員身分

訪問 LINE Developers，點擊 Log in 並使用您的 LINE 帳號登入。點擊 Create 新增一個 Provider。

2

Step 2: 建立通訊頻道

選擇「Create a Messaging API channel」。填寫機器人的顯示名稱、大頭照與常用的信箱。

3

Step 3: 提取對接數據

在 Channel settings 分頁，找到 Channel secret 並複製。在 Messaging API 分頁，拉到最底點擊 Issue 獲取 Channel access token。

4. 龍蝦故障排除百科全書

401 Unauthorized

【你看到了什麼】 龍蝦回覆："API Key is invalid"。

【發生什麼事】 您的鑰匙過期了，或是複製時少了一個字。

【10 秒鐘修復】 登入控制台，刪除舊 Key 並建立一個新的，重新傳送給龍蝦。

403 Forbidden

【你看到了什麼】 報錯 "Access restricted to specific regions"。

【發生什麼事】 該 AI 模型不支援您目前的連線地區（如：Gemini 封鎖某些 IP）。

【10 秒鐘修復】 龍蝦雲用戶免驚。請私訊後台，我們會為您一鍵切換至「美區極速節點」。

429 Rate Limit

【你看到了什麼】 顯示 "Wait for 1 minute before retrying"。

【發生什麼事】 機器人太忙了，或是免費版的頻率太低。

【10 秒鐘修復】 這是提示您該去 OpenAI 充值 $5U 成為「付費用戶」了，這能提升 100 倍速度。
